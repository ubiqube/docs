== Overview

This tutorial details the IMPORT object implementation in the case of a
simple variable array on a CLI based device. The following example uses
a Cisco configuration sample.

== Configuration Sample

Here is an example of the configuration that you want to import in your
object.

....
### DHCP helper-address ###
interface FastEthernet0
  ip helper-address 1.2.3.4
  ip helper-address 2.3.4.5
....
